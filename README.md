# Earthquake-Prediction-LightGBM
Acoustic Signal Analysis & Prediction (LANL Earthquake Reproduction)
Даний проєкт присвячений дослідженню та репродукції переможного рішення (1st place) змагання Kaggle LANL Earthquake Prediction. Основна мета — прогнозування часу до руйнування (Time to Failure) на основі високочастотних акустичних сигналів.

🚀 Чому це важливо для Defense Tech?
Алгоритми, представлені в цьому проєкті, мають пряме застосування в мілітарних технологіях:
1.Акустична детекція: Виявлення та класифікація звукових сигнатур техніки (дронів, бронетехніки).
2.Predictive Maintenance: Прогнозування виходу з ладу механічних вузлів БпЛА на основі аналізу вібрацій.
3.Сейсмічний моніторинг: Аналіз коливань ґрунту для виявлення переміщення важкої техніки.
🛠 Технічний стек
1.Language: Python 3.
2.xML Model: LightGBM (Gradient Boosting)
3.Signal Processing: NumPy, SciPy (FFT, Statistical Analysis)Visualization: Matplotlib, Seaborn🧠 Методологія (Feature Engineering)Оскільки акустичний сигнал є надзвичайно шумним, ключовим етапом є вилучення статистичних ознак з вікон сигналу. Для кожного сегменту обчислюються:Статистика в часовій області: Mean, Standard Deviation, Quantiles.Спектральний аналіз: Швидке перетворення Фур'є (FFT) для аналізу частотних характеристик.$$S_{xx}(f) = |X(f)|^2$$Rolling Statistics: Аналіз зміни характеристик всередині часових вікон.📈 РезультатиВикористання моделі LightGBM дозволило досягти високої точності прогнозування критичних моментів руйнування, мінімізуючи середню абсолютну помилку (MAE).
